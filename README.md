# SRBD Coding Round Solutions

This repository contains my solutions to various programming problems that appeared in the **SRBD coding round**. The goal of this repository is to document my approaches, practice problem-solving techniques, and maintain a reference for algorithms and coding patterns used during the contest.

## 📌 Overview

The repository includes implementations of different algorithmic problems solved using **C++**.  
The problems cover a variety of topics commonly tested in technical coding rounds.

Some of the main concepts used include:

- Backtracking
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Recursion
- Graph Traversal
- Greedy Techniques
- Brute Force Approaches
- Optimization Techniques
